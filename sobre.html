O Gerador de Imagens com Inteligência Artificial é um projeto educacional desenvolvido com o objetivo de aplicar, de forma prática, os conhecimentos adquiridos durante a disciplina. Ele permite que os usuários criem imagens digitais com base em descrições de texto, utilizando uma API de IA.

Através dessa aplicação, os alunos praticam conceitos fundamentais de desenvolvimento web, como estruturação com HTML, estilização com CSS, interatividade com JavaScript e integração com serviços externos.

O projeto estimula a criatividade, o trabalho em equipe e a resolução de problemas, além de aproximar os estudantes das tecnologias modernas que estão sendo usadas no mercado atual.
