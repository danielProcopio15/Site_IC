Bem-vindo ao nosso Gerador de Imagens com Inteligência Artificial! Este projeto foi desenvolvido como parte das práticas da disciplina, com o objetivo de explorar e aplicar conceitos fundamentais de desenvolvimento web, integração com APIs e uso de inteligência artificial para geração de imagens. A proposta é simples: permitir que o usuário digite uma descrição de uma imagem, como por exemplo "um gato astronauta no espaço", e com isso, o sistema utiliza uma API de IA para gerar automaticamente uma imagem baseada nesse texto. É uma forma divertida e educativa de entender como tecnologias modernas podem transformar ideias em conteúdo visual. Durante o projeto, você terá contato com várias tecnologias: HTML para estruturar as páginas, CSS para criar um visual moderno e responsivo, JavaScript para integrar com APIs e tornar o site interativo. O que você encontrará neste site? Página Inicial: Uma introdução geral ao projeto (essa que você está lendo agora). Gerar Imagem: Onde você poderá inserir sua descrição e gerar imagens com IA. Galeria: Exibe as imagens geradas pelos usuários. Sobre: Explica os objetivos do projeto e apresenta a equipe envolvida. Contato: Um formulário para sugestões, dúvidas ou feedback.
